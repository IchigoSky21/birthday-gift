## 🌟 Tentang Proyek

**KadoTepat** adalah aplikasi web interaktif yang memanfaatkan kecerdasan buatan (**Google Gemini API**) untuk memberikan rekomendasi kado ulang tahun berdasarkan preferensi penerima, seperti hobi, usia, hubungan, hingga anggaran (*budget*). 

Dibangun dengan pendekatan *frontend-only* menggunakan Vanilla JavaScript dan Tailwind CSS, aplikasi ini ringan, cepat, dan terintegrasi langsung dengan penyimpanan lokal (*localStorage*) serta fitur berbagi via WhatsApp.

---

## ✨ Fitur Utama

- **🤖 AI-Powered Recommendation:** Menghasilkan ide kado unik dan spesifik menggunakan Google Gemini API.
- **🎯 Filter Kustom:** Sesuaikan rekomendasi berdasarkan kategori usia, hobi, jenis hubungan, dan kisaran harga.
- **⭐ Daftar Favorit:** Simpan ide kado pilihan Anda langsung di browser menggunakan `localStorage`.
- **💬 Bagikan ke WhatsApp:** Kirim hasil rekomendasi kado langsung ke teman atau keluarga dengan satu klik.
- **⚡ Responsive & Modern UI:** Tampilan antarmuka yang bersih dan ramah pengguna berkat Tailwind CSS.
